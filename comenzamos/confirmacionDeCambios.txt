confirmacion de cambios

Con estos comandos vamos a poder confirmar los cambios que estamos haciendo en nuestro repositorio local

1- dir:
   Nos permite listar todos los archivos dentro de una carpeta.
2- git init: 
   Nos permite crear un repositorio local en nuestra computadora.
3- git add .
   Nos permite agregar archivos y carpetas al repositorio local.
4- git status
   Nos permite listar todos los archivos y carpetas dentro del repositorio local.
5- git commit -m "Mensaje"
   Nos permite asegurar que los archivos esten listos para pasar al repositorio remoto junto a un mensaje.
6- git look 
   Nos permite saber que cosas se icieron en este repositorio, nos muestra cosas como
   commits,
   nombre de quien hizo el o los commits,
   fecha,
   mensaje dl commits,
   nombre de la rama,
